Aether Widget Demo

This page demonstrates the embedded Aether chat widget. Click the 💬 button in the corner to start chatting.

<!-- Add this to your website -->
<script src="widget/aether-widget.js" data-aether-host="http://localhost:5173" data-aether-tenant="demo" data-color="#1a73e8"></script>

Configuration Options

Set these data-* attributes on the script tag:

  • data-aether-host — backend URL (required)
  • data-aether-tenant — tenant slug (default: "demo")
  • data-color — primary color (default: #1a73e8)
  • data-position — "bottom-right", "bottom-left", "top-right", "top-left"
  • data-greeting — first message text
  • data-auto-open — "true" to open on load

JavaScript API

AetherWidget.open()       // Open the widget
AetherWidget.close()      // Close it
AetherWidget.send("text") // Send a message
AetherWidget.setToken("jwt_token")  // Set auth token
AetherWidget.isOpen()     // Check if open
AetherWidget.isConnected() // Check WebSocket status
